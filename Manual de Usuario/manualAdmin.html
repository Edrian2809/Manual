1. Función del Administrador

El usuario con perfil Administrador en SERPROM Gestión tiene acceso completo a todas las funciones del sistema. Es responsable de:

  • Crear, modificar y eliminar registros (clientes, técnicos, equipos).
  • Supervisar y asignar órdenes de trabajo.
  • Controlar el flujo de facturación y pagos.
  • Revisar reportes operativos y estadísticas.
  • Gestionar permisos y usuarios del sistema.

2. Gestión de Usuarios

Desde el módulo de usuarios, el administrador puede:

  • Registrar nuevos usuarios (técnicos, operadores).
  • Asignar roles (técnico, administrativo, cliente).
  • Editar contraseñas o datos personales.
  • Desactivar cuentas temporalmente.

3. Control de Órdenes

Puede crear nuevas órdenes de trabajo, asignar técnicos, definir fechas, estados, y marcar el tipo de servicio: preventivo, correctivo, instalación, etc.

El panel muestra un historial completo de cada orden con detalles de ejecución y observaciones.

4. Mantenimiento de Equipos

En la vista de mantenimientos, puede:

  • Programar mantenimientos preventivos.
  • Registrar mantenimientos correctivos.
  • Ver histórico por equipo, técnico o cliente.

5. Gestión de Facturación

Desde el módulo de ventas puede:

  • Emitir facturas de servicios.
  • Registrar pagos recibidos y comprobantes.

6. Reportes y Estadísticas

Se puede acceder a:

  • Reportes filtrados por cliente, técnico, fecha o tipo de servicio.

7. Seguridad y Roles

El administrador tiene la posibilidad de:

  • Definir roles personalizados.
  • Configurar niveles de acceso a cada módulo.
  • Visualizar logs o historial de acceso al sistema.

8. Recomendaciones

  • Respaldar la base de datos semanalmente.
  • Revisar reportes y desempeño de técnicos al cierre de cada mes.
  • Actualizar las credenciales de acceso regularmente.
